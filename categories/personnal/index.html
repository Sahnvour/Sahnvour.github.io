Dalicious

19/Nov 2014 1 min. read

Projects Personnal

In november 2014, I registered for the Game Challenge, taking place during the Montpellier In Game summit. Teams of around 10 students were formed with developers, graphists, game designers and sound artists. Each group had to draw five random constraints for their game. We picked the following: Use a camera as input for the game Use a 2D axis input for something else than moving a camera or an object in-game Have a cooking mechanic as core gameplay Take inspiration from Dali’s work for the game art Have a dynamic soundtrack Not an easy task.

Tweetable mathematical art: Julia sets

3/Aug 2014 2 min. read

Projects Personnal

This post is my first entry to this challenge on Stackexchange’s PPCG: Tweetable mathematical art. Julia sets If there’s a Mandelbrot, there should be a Julia set too. You can spend hours tweaking the parameters and functions, so this is just a quick one that looks decent. Inspired from Martin’s participation. unsigned short red_fn(int i, int j){ #define D(x) (x-DIM/2.)/(DIM/2.) float x=D(i),y=D(j),X,Y,n=0;while(n++<200&&(X=x*x)+(Y=y*y)<4){x=X-Y+.36237;y=2*x*y+.32;}return log(n)*256;} unsigned short green_fn(int i, int j){ float x=D(i),y=D(j),X,Y,n=0;while(n++<200&&(x*x+y*y)<4){X=x;Y=y;x=X*X-Y*Y+-.7;y=2*X*Y+.27015;}return log(n)*128;} unsigned short blue_fn(int i, int j){ float x=D(i),y=D(j),X,Y,n=0;while(n++<600&&(x*x+y*y)<4){X=x;Y=y;x=X*X-Y*Y+.36237;y=2*X*Y+.32;}return log(n)*128;} Would you like some RNG?
