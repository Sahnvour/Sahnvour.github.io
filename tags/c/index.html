Dalicious

19/Nov 2014 1 min. read

Projects Personnal

In november 2014, I registered for the Game Challenge, taking place during the Montpellier In Game summit. Teams of around 10 students were formed with developers, graphists, game designers and sound artists. Each group had to draw five random constraints for their game. We picked the following: Use a camera as input for the game Use a 2D axis input for something else than moving a camera or an object in-game Have a cooking mechanic as core gameplay Take inspiration from Dali’s work for the game art Have a dynamic soundtrack Not an easy task.

You can spend hours tweaking the parameters and functions, so this is just a quick one that looks decent. 29/May 2014 1 min. read

Projects University

As an optional project using innovative technologies, we (a group of 7 students) created a robot with the following objectives: - grab and store small plastic balls - differentiate blue and red balls - find its way in an environment with obstacles Setup The robot was built with a Mindstorm brick to control the different motors (wheels, arms). We put a smartphone on it, filming the ground just in front of it to detect balls and move accordingly.
